Privacy Policy

Last updated: March 24, 2026

Easy Card Offers is a browser extension that helps users inspect and activate eligible bank offers on supported bank pages.

What the Extension Does

The extension:

  • checks the active page to see whether it matches supported offer surfaces
  • reads page title, URL, and local page DOM markers to detect supported bank offers
  • collects visible offer candidates from supported pages
  • runs user triggered activation flows on supported pages
  • stores lightweight local state so popup results and batch runs can be restored

Data We Access

When you use the extension on supported pages, it may access:

  • the current tab URL
  • the current page title
  • the page DOM needed to detect supported offers and activation state
  • local extension storage used for popup summaries and batch run state

Data We Do Not Collect or Sell

The extension does not:

  • send browsing data to remote servers
  • use analytics or tracking pixels
  • sell personal information
  • share personal information with advertisers or data brokers

Local Storage Only

Any stored state is kept locally in the browser's extension storage and is used only to:

  • restore popup results
  • restore Chase and Bank of America batch run summaries
  • keep batch state tied to the current page

Permissions

The extension currently uses:

  • tabs to inspect the active tab URL for page aware popup and batch state behavior
  • storage to save local popup and batch state

Your Choices

You can stop all extension activity at any time by:

  • closing the popup without starting a flow
  • uninstalling the extension
  • clearing browser extension storage

Contact

For support or privacy questions, use the support contact published with the extension listing.
